# DataSciBot – Custom GPT

## Overview

DataSciBot is a custom GPT built to help beginners understand data science concepts, improve machine learning models, and get project guidance.

## Features

* Explains data science concepts
* Helps with ML models and projects
* Provides learning resources

## Example Prompts & Responses

**Prompt:** Explain overfitting in simple terms
**Response:**
Overfitting happens when a model memorizes the training data instead of learning the real pattern. It performs well on training data but poorly on new data.
